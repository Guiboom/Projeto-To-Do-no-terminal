# Projeto-To-Do-no-terminal

Aplicação de gerenciamento de tarefas desenvolvida em Python para execução diretamente no terminal.

## Sobre o Projeto

O Projeto-To-Do-no-terminal foi criado com o objetivo de praticar conceitos fundamentais de programação através do desenvolvimento de uma aplicação CRUD (Create, Read, Update e Delete).

O sistema permite adicionar, listar, concluir e remover tarefas, além de armazenar os dados localmente utilizando arquivos JSON para manter as informações entre diferentes execuções do programa.

## Funcionalidades

* Adicionar tarefas
* Listar tarefas cadastradas
* Marcar tarefas como concluídas
* Remover tarefas
* Definir prioridade das tarefas
* Registro automático de data e hora
* Salvamento automático em arquivo JSON
* Interface interativa via terminal
